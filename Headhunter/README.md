Triagem de Candidatos (Pandas):
Criei este script para resolver um problema clássico de RH: a perda de tempo filtrando currículos que não batem com a vaga. Em vez de abrir planilha por planilha, o script processa tudo e me entrega quem realmente importa.

O que o script faz:
Limpeza: Ele não aceita dados sujos. Usei Pandas para garantir que nomes e skills estejam padronizados.

Ranking:
Não é só um filtro comum. Criei uma lógica de pontuação onde Python e SQL têm pesos maiores, jogando os melhores candidatos para o topo da lista.

Saída:
O resultado é um arquivo limpo, pronto para o recrutador ligar para o candidato.

Ferramentas:
Python + Pandas
